给你完整梳理一下现在所有可用的 YAML 参数，分三层：

---

## 第一层：基础参数（所有引擎通用）

```yaml
- index: 5 # 元素编号（必填）
  action:
    type: click # 操作类型
    on_not_found: fail # 找不到时: fail(终止) / skip(跳过)
    wait_for_element: false # 是否等待元素出现
    wait_timeout: 10 # 等待超时（秒）
    wait_before: 0.5 # 操作前等待（秒）
    wait_after: 0.5 # 操作后等待（秒）
    wait_strategy: timeout # timeout / navigation / element_appear
    # fill 特有:
    value: "${ACCOUNT}" # 填写内容/变量
    clear_first: true # 填前清空
    # screenshot 特有:
    screenshot_name: "step_5" # 截图文件名
```

## 第二层：条件执行（IF）

```yaml
- index: 6
  if: # ← 条件
    condition: element_exists #   element_exists / element_not_exists
    ref_index: 5 #   引用元素的编号
  action:
    type: click
```

## 第三层：高级引擎参数（`advanced_runner.py` 专用）

```yaml
- index: 6
  action:
    type: click
    on_not_found: skip
  _config: # ← 高级配置（advanced_runner.py 识别）
    retry: # 重试策略
      max_retries: 8 #   最大重试次数（默认 5）
      base_delay: 2.0 #   首次重试等待秒数（默认 1.0）
      max_delay: 30.0 #   最大等待秒数（默认 30）
      backoff_factor: 2.0 #   指数退避倍数（默认 2.0）
      jitter: true #   是否随机抖动防碰撞（默认 true）
      on_exhausted: human #   重试用尽后: retries继续/skip跳过/fail失败/human人工介入
    human: # 人工介入配置
      enabled: true #   是否启用人工介入（默认 true）
      timeout: 300 #   等待人工操作的秒数（默认 300）
```

## 你贴的那段代码的意思

```yaml
- index: 6 # 第 6 号元素
  action:
    type: click # 执行点击操作
    on_not_found: skip # 找不到就跳过（不终止流程）
  _config:
    retry:
      max_retries: 8 # 最多重试 8 次（比默认多）
      base_delay: 2.0 # 首次等 2 秒（比默认慢）
      on_exhausted: human # 8 次都找不到 → 弹出人工介入
    human:
      timeout: 300 # 等你 5 分钟来手动处理
```

## 完整参数表

| 层级     | 参数                           | 可选值                                                                               | 默认值  | 说明               |
| -------- | ------------------------------ | ------------------------------------------------------------------------------------ | ------- | ------------------ |
| **基础** | `type`                         | click/fill/check_exists/check_visible/wait_visible/hover/select/scroll_to/screenshot | click   | 操作类型           |
|          | `on_not_found`                 | fail/skip                                                                            | fail    | 找不到时行为       |
|          | `wait_before`                  | 数字                                                                                 | 0.5     | 操作前等待秒数     |
|          | `wait_after`                   | 数字                                                                                 | 0.5     | 操作后等待秒数     |
|          | `wait_strategy`                | timeout/navigation/element_appear                                                    | timeout | 等待策略           |
|          | `wait_for_element`             | true/false                                                                           | false   | 是否等待元素出现   |
|          | `wait_timeout`                 | 数字                                                                                 | 10      | 等待超时秒数       |
|          | `value`                        | 字符串                                                                               | —       | 填写内容/`${变量}` |
|          | `clear_first`                  | true/false                                                                           | true    | 填前清空           |
| **IF**   | `if.condition`                 | element_exists/element_not_exists                                                    | —       | 条件类型           |
|          | `if.ref_index`                 | 数字                                                                                 | —       | 引用元素编号       |
| **高级** | `_config.retry.max_retries`    | 数字                                                                                 | 5       | 最大重试次数       |
|          | `_config.retry.base_delay`     | 数字                                                                                 | 1.0     | 首次重试等待秒数   |
|          | `_config.retry.max_delay`      | 数字                                                                                 | 30.0    | 最大等待秒数       |
|          | `_config.retry.backoff_factor` | 数字                                                                                 | 2.0     | 指数退避倍数       |
|          | `_config.retry.jitter`         | true/false                                                                           | true    | 随机抖动           |
|          | `_config.retry.on_exhausted`   | retries/skip/fail/human                                                              | skip    | 重试用尽后行为     |
|          | `_config.human.enabled`        | true/false                                                                           | true    | 启用人工介入       |
|          | `_config.human.timeout`        | 数字（秒）                                                                           | 300     | 等待人工操作超时   |

注意：`_config` 层目前只在 `advanced_runner.py` 生效，旧的 `run_custom_steps.py` 和 `run_flow_steps.py` 不认识它，会忽略。如果你想所有引擎都支持，我可以把 `_config` 提取到通用模块。
